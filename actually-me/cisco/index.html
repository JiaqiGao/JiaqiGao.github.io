Cisco Industrial Network Interactive Visualizations
Looking at a visualization that encapsulates so much flow data at once can be daunting.
Let's examine the data first by where there is communication, then by how much
communication is happening measured by number of packets.


OVERVIEW


Here each blue line represents a flow between modules.
Opacity is being used to represent number of flows.





RECEIVE


Black circles indicate modules that did not receive any packets.
Size of red ones correspond to relative number of packets received (using log scale).
Click on a circle to see the modules that sent packets to the selected module.





SEND


Black circles indicate modules that did not send any packets.
Size of blue ones correspond to relative number of packets sent (using log scale).
Click on a circle to see the modules that received the packets sent by the selected module.
